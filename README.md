# 🎓 Projet de Programmation Web - Clone de Site Scolaire

Ce projet consiste en la création d'un site web moderne et interactif, 
fortement inspiré de l'interface et de l'expérience utilisateur de mon établissement scolaire. 
L'objectif est de mettre en pratique les fondamentaux du développement Front-End.

## 📌 Aperçu du Projet
Le site reprend les sections clés d'un portail éducatif : 
  accueil, présentation des formations, et outils pour les étudiants.

## 🛠️ Technologies Utilisées
- **HTML5** : Structure sémantique des pages.
- **CSS3** : Mise en page (Flexbox/Grid) et design responsive.
- **JavaScript** : Interactivité et manipulation dynamique du DOM.
- **JSON** : Stockage et récupération de données structurées.

## 📂 Structure du Dépôt
L'organisation des fichiers suit les bonnes pratiques de développement web :

- `/html` : Contient l'ensemble des pages du site.
- `/css` : Feuilles de style pour le design et la typographie.
- `/js` : Scripts pour les fonctionnalités interactives.
- `/json` : Fichiers de données (utilisés pour simuler une base de données ou du contenu dynamique).
- `/img` : Ressources graphiques et icônes.
- `Maquette ProjetWeb.pdf` : Le document de conception original servant de référence.
